# chess_game
A chessgame that can be run in your IDE


# Jeu d'Échecs en Python / Chess Game in Python

Ce projet est une implémentation simple d'un jeu d'échecs en Python.  
This project is a simple implementation of a chess game in Python.

---

## Fonctionnalités / Features

- Plateau 8x8 représenté par une liste de listes.  
  8x8 board represented as a list of lists.  
- Placement automatique des pièces blanches et noires.  
  Automatic placement of white and black pieces.  
- Affichage du plateau dans la console.  
  Display of the board in the console.  
- Déplacement des pièces via des entrées utilisateurs (ex: C2 → C3).  
  Move pieces via user input (e.g., C2 → C3).  
- Quitter le jeu en tapant `Q`.  
  Quit the game by typing `Q`.
